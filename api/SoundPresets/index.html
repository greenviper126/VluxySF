SoundPresets

A collection of preset functions for applying common sound effects, such as jitter, to Sound instances. These presets can be easily applied to sounds to achieve dynamic and natural variations in sound playback.

Functions​

NaturalJitter​

</>
SoundPresets.NaturalJitter(
center: number?,--

The central value around which the jitter will occur (default: 1).

spread: number?,--

The maximum deviation from the center value (default: 0.1).

parent: Instance?--

The parent instance to which the sound will be attached after applying the preset.

) → (Sound) → ()--

A preset function that applies natural jitter to a sound's volume and playback speed, creating more dynamic and less repetitive audio effects.

applys a unitLog jitter to volume and pitch with an option to set the parent.
